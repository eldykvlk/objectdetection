# cara pemakaian (How to Use)


1. Download repository
```
Git clone https://github.com/eldykvlk/objectdetection
```

2. Membuat direktori
```
cd objectdetection
```

3. Install Library OpenCV
```
pip install opencv-contrib-python
```

4. Install Numpy
```
pip install numpy
```


5. Run Project
```
python namafile.py
```
